## Fingerprinting Module

This folder contains the fingerprint comparison algorithm from the FingerprintComparer project.

### Files Included

- **FingerprintComparer.dll** - Main fingerprint comparison logic
- **SourceAFIS.dll** - Core fingerprint recognition algorithm
- **SixLabors.ImageSharp.dll** - Image processing library
- **Dahomey.Cbor.dll** - CBOR serialization support
- **System.IO.Pipelines.dll** - I/O abstractions
- **FingerprintComparer.deps.json** - Dependency manifest
- **SourceAFIS.pdb & .xml** - Debugging symbols and documentation
- **LICENSE** - Apache 2.0 license
